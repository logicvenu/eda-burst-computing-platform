# Cloud-Agnostic EDA Burst Computing Platform

## Overview

A hybrid HPC burst computing platform enabling semiconductor organizations to elastically extend on-premises Electronic Design Automation (EDA) workloads into public cloud infrastructure.

Supported Clouds:

- AWS
- Google Cloud Platform

Core Capabilities:

- HPC Cloud Bursting
- Slurm / LSF Scheduling
- Shared Storage
- Elastic Compute Scaling
- Analytics Platform
- AI-Driven Optimization
- Generative AI Assistant
- Agentic HPC Operations
