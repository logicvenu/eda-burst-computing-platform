# Cloud-Agnostic EDA Burst Computing Platform

## Overview

The Cloud-Agnostic EDA Burst Computing Platform enables semiconductor organizations to elastically extend on-premises Electronic Design Automation (EDA) workloads into public cloud infrastructure.

The platform supports hybrid High-Performance Computing (HPC) environments and enables cloud bursting into AWS and Google Cloud Platform while maintaining existing engineering workflows, license management systems, and storage architectures.
